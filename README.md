# 提示词画廊 · Prompt Gallery

一个面向设计师、创作者和 AI 产品开发者的网页设计提示词灵感库。

它收录了 177 个 MotionSites 风格的网页设计卡片，其中 64 个带完整提示词。每张卡片都对应一种真实网页方向，比如 SaaS、Agency、Portfolio、Hero Section、Landing Page、3D Website 等。你可以先看视觉风格，再一键复制完整提示词，丢给 AI 生成同款网页。

## 适合谁

- 想快速做出高质量网页首屏的人
- 需要设计参考和提示词模板的 AI 产品开发者
- 想学习高级网页视觉表达的设计师
- 想把 MotionSites 风格拆成可复用提示词的人

## 文件说明

- `motionsites-index.html`：提示词画廊主页，包含 177 个设计卡片、分类筛选、搜索、复制提示词和实时预览。
- `previews/liquid-glass-agency.html`：精选案例的实时网页预览。
- `index.html`：Hero 提示词工坊，用于根据页面类型和内容生成更完整的网页提示词。

## 本地打开

直接用浏览器打开：

```bash
open /Users/hanwuyue/prompt-builder/motionsites-index.html
```

## 项目价值

这个项目不是简单收藏网页截图，而是把“好看的网页风格”拆成可以复用的提示词资产。它适合用来做设计灵感库、AI 生成网页的起点，也可以作为后续产品化的 prompt builder 原型。
